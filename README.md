# opticalMarket_backend

Backend API for the VNW Optical Marketplace platform.

## Stack
- Node.js
- TypeScript
- NestJS
- PostgreSQL (Supabase)
- Prisma ORM

## Requirements
- Node.js >= 18
- npm (or yarn/pnpm)
- PostgreSQL connection (Supabase)
